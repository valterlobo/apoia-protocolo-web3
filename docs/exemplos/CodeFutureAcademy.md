# 💻 Projeto de Financiamento Coletivo — Formação em Programação para Jovens da Rede Pública

# 1. Visão Geral do Projeto

## Nome do Projeto

**Code Future Academy**

## Descrição do Projeto

O Code Future Academy é um programa educacional de impacto social voltado ao ensino de programação para jovens entre 14 e 16 anos matriculados em escolas públicas. O projeto terá duração de 3 meses e atenderá 100 alunos divididos em 4 turmas de 25 estudantes.

A iniciativa busca democratizar o acesso à tecnologia e capacitar jovens em lógica de programação, desenvolvimento de software e pensamento computacional, preparando-os para oportunidades futuras no mercado digital.

A seleção dos participantes será realizada em escolas públicas por meio de um processo seletivo com teste básico de lógica e matemática.

O projeto contará com computadores fornecidos em modelo de leasing, além de uma equipe composta por 4 instrutores e 4 estagiários para suporte pedagógico e acompanhamento extra aula.

# 20. Conclusão

Além da formação técnica, o programa pretende despertar nos estudantes confiança, criatividade e novas perspectivas profissionais.
